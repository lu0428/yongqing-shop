# 詠晴官網商品素材整理說明

## 資料夾結構

```
yungching-shop/
├── products/
│   ├── 中西藥局/products.json   ← 填入詠晴中西藥局的商品資料
│   └── 漢方/products.json       ← 填入詠晴漢方的商品資料
├── images/
│   ├── 中西藥局/                ← 放中西藥局商品圖（jpg/png）
│   └── 漢方/                   ← 放漢方商品圖（jpg/png）
└── assets/                     ← Logo、Banner 等品牌素材
```

---

## 你需要準備的素材

### 每個商品需要填入：

| 欄位 | 說明 | 範例 |
|------|------|------|
| id | 商品代號（自訂，不重複） | `zx001` |
| name | 商品名稱 | `維他命C 500mg` |
| category | 分類 | `維他命` |
| price | 售價（數字） | `299` |
| originalPrice | 原價（如無折扣填同售價） | `350` |
| description | 商品說明（1～2句） | `高單位維他命C，促進膠原蛋白合成` |
| image | 圖片路徑（放進 images 資料夾後填） | `images/中西藥局/vitc.jpg` |
| inStock | 是否有庫存 | `true` 或 `false` |
| tags | 標籤（可多個） | `["保健", "維他命"]` |

### 圖片建議規格
- 格式：JPG 或 PNG
- 尺寸：正方形為佳，建議 800x800px 以上
- 命名：用英文或數字，避免空格（例：vitc_500.jpg）
